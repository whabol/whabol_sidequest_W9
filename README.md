## Description

This p5.js project uses week 9 example 3 and I have iterated it to include a debug menu/ screen that pops up when the "g" key is pressed. On the debug menu the option to toggle moon gravity on/off is enabled as 
well as the option to toggle slow mode on/off, what keys to press for the toggling is also stated on the pop up screen. To exit the debug menu, clicking "g" returns you to the game.
